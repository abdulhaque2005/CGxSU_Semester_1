# Basic CRUD Routes

| Method | Endpoint | Description |
| :--- | :--- | :--- |
| **GET** | `/reviews` | Fetch all review records |
| **GET** | `/reviews/:reviewId` | Fetch single review using ID |
| **POST** | `/reviews` | Create new review |
| **PUT** | `/reviews/:reviewId` | Replace complete review |
| **PATCH** | `/reviews/:reviewId` | Update review fields |
| **DELETE** | `/reviews/:reviewId` | Delete review record |
| **GET** | `/bookings` | Fetch all booking records |

| **DELETE** | `/bookings/:bookingId` | Delete booking |
| **GET** | `/conflicts` | Fetch all conflict records |
| **GET** | `/conflicts/:conflictId` | Fetch conflict using ID |
| **POST** | `/conflicts` | Create new conflict |
| **PUT** | `/conflicts/:conflictId` | Replace conflict record |
| **PATCH** | `/conflicts/:conflictId` | Update conflict fields |
| **DELETE** | `/conflicts/:conflictId` | Delete conflict |
| **GET** | `/users` | Fetch all users |
| **GET** | `/users/:userId` | Fetch user using ID |
